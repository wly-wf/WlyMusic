# Win11音乐播放器UI布局设计 v1.0

## 设计状态
✅ **已确认决策**:
1. 整体布局：左侧导航栏 + 主内容区 + 底部播放控制栏
2. 歌词显示：点击底部歌词按钮，主内容区切换为歌词视图
3. 标题栏：显示"WlyMusic"和当前页面标题
4. 导航结构：左侧垂直导航栏，包含音乐库、歌单管理
5. 内容展示：列表视图为主，支持排序搜索

## 详细设计

### 1. 标题栏设计
**实现**: Electron `titleBarOverlay` 创建Win11原生风格标题栏
**左侧显示**: 应用图标 + "WlyMusic" + " > " + 当前页面标题（如"全部歌曲"）
**右侧按钮**: 设置⚙️、最小化、最大化/还原、关闭
**交互**: 可拖动移动窗口，双击切换最大化，点击设置图标打开设置页面

### 2. 左侧导航栏 (240px宽度)
**背景**: 亚克力透明效果，右侧圆角
**导航项目**:
- **音乐库**
  - 全部歌曲
  - 艺术家（按艺术家分组）
  - 专辑（按专辑分组，显示封面）
  - 最近播放
- **歌单管理**
  - 我的歌单1（用户创建）
  - 我的歌单2
  - [+ 新建歌单] 按钮

**交互**:
- 点击导航项切换主内容区
- 歌单可展开/折叠
- 当前选中项高亮显示
- 新建歌单弹出输入框

### 3. 主内容区
**两种显示模式**:
1. **歌曲列表模式** (默认)
2. **歌词视图模式** (点击歌词按钮切换)

#### 3.1 歌曲列表视图
```
┌─────────────────────────────────────────────────────┐
│ [搜索框]      [排序: 名称↑] [显示: 列表] [封面: 显示] │
├──┬────────┬─────────┬─────────┬───────┬────────────┤
│  │ 标题   │ 艺术家  │ 专辑    │ 时长  │ 操作       │
├──┼────────┼─────────┼─────────┼───────┼────────────┤
│◯ │ 歌曲1  │ 艺术家1 │ 专辑1   │ 3:45  │ ⋮ (菜单)   │
├──┼────────┼─────────┼─────────┼───────┼────────────┤
│◯ │ 歌曲2  │ 艺术家2 │ 专辑2   │ 4:20  │ ⋮ (菜单)   │
└──┴────────┴─────────┴─────────┴───────┴────────────┘
```

**列表功能**:
- **复选框**: 多选歌曲进行批量操作
- **表头排序**: 点击标题、艺术家、专辑、时长排序
- **操作菜单** (⋮):
  - 播放
  - 添加到歌单
  - 查看歌词（在歌词视图显示）
  - 歌曲信息
- **搜索框**: 实时过滤歌曲
- **视图控制**:
  - 排序方式下拉菜单
  - 显示模式切换（列表/网格）- v1.1
  - 封面显示开关（✅ 已确定：显示小封面，32x32px）

#### 3.2 歌词视图模式
**触发方式**: 点击底部控制栏歌词按钮 🎵
**布局**:
```
┌─────────────────────────────────────────────────────┐
│ [← 返回歌曲列表]  歌曲标题 - 艺术家                 │
├─────────────────────────────────────────────────────┤
│                                                     │
│                歌词内容区域                         │
│       当前播放行高亮，自动滚动居中                  │
│                                                     │
├─────────────────────────────────────────────────────┤
│ 歌词来源：内嵌标签 | 字体大小：中 🔽 | 同步：开      │
└─────────────────────────────────────────────────────┘
```

**歌词视图功能**:
- **同步滚动**: 根据播放时间自动滚动，高亮当前行
- **字体控制**: ✅ v1.0支持小/中/大三种字体大小
- **来源显示**: 显示歌词来源（内嵌ID3、外部.lrc）
- **无歌词处理**: 显示"暂无歌词"提示，提供搜索按钮（v1.1）
- **返回按钮**: 点击返回歌曲列表，或再次点击歌词按钮返回

### 4. 底部播放控制栏 (80px高度)
**布局分区**:
```
[专辑封面64x64] [歌曲信息] [播放控制] [其他控制]
```

**详细组件**:
1. **专辑封面区域** (左侧)
   - 当前歌曲专辑封面（64x64px，圆角）
   - 点击也可切换歌词视图（备用交互）

2. **歌曲信息区域** (左中)
   - 歌曲标题（加粗，截断显示）
   - 艺术家 - 专辑（次级文本）
   - 点击可跳转到歌曲在列表中的位置

3. **播放控制区域** (中部)
   - 上一首 ◀
   - 播放/暂停 ⏸ (状态切换)
   - 下一首 ▶
   - 进度条 [────●────] 可拖拽
   - 时间显示 当前/总时长

4. **其他控制区域** (右侧)
   - 音量控制 🔊 (滑动条或点击静音)
   - 播放模式 ♻ (顺序/随机/单曲循环/列表循环)
   - 播放列表 ⏻ (显示当前队列)
   - **歌词按钮 🎵** (切换主内容区歌词视图，高亮表示激活)

## 待定问题解决方案

### 1. 迷你播放器模式
**决策**: v1.0 **不实现** 迷你播放器模式
**理由**: MVP聚焦核心功能，迷你模式属于增强功能
**v1.1计划**: 双击标题栏或拖出播放控制栏创建迷你窗口

### 2. 专辑封面显示
**决策**: 列表视图 **显示小封面** (32x32px)
**位置**: 歌曲列表第一列，复选框右侧
**优势**: 视觉更丰富，易于识别专辑
**实现**: 从ID3标签读取，无封面时显示默认图标

### 3. 歌词字体控制
**决策**: v1.0 **支持基础字体大小调整**
**选项**: 小、中、大 三种预设大小
**位置**: 歌词视图底部控制栏
**存储**: 保存到用户设置，应用重启后保持

### 4. 歌词搜索功能
**决策**: v1.1实现网络歌词搜索
**v1.0方案**: 无歌词时显示"暂无歌词"和"搜索歌词"按钮（点击无操作，v1.1实现）

### 5. 设置功能位置
**决策**: v1.0设置功能位于标题栏右上角（设置⚙️图标），点击打开设置页面
**v1.0设置内容**: 音乐库目录管理、歌词字体大小设置、应用重启后生效的配置
**v1.1计划**: 主题切换、快捷键设置、高级音频设置

## 状态管理设计

### 全局状态 (Zustand Store)
```typescript
interface AppState {
  // 播放状态
  currentSong: Song | null;
  isPlaying: boolean;
  progress: number;
  volume: number;
  playMode: 'sequential' | 'random' | 'repeat-one' | 'repeat-all';
  
  // 界面状态
  activeView: 'library' | 'playlist' | 'lyrics';
  activeNavItem: string;
  showLyricsView: boolean;
  
  // 音乐库
  musicLibrary: Song[];
  playlists: Playlist[];
  selectedSongIds: string[];
  
  // 用户设置
  settings: {
    lyricFontSize: 'small' | 'medium' | 'large';
    showCoverInList: boolean;
    theme: 'light' | 'dark' | 'system';
  };
}
```

### 路由/视图状态
- `activeView`: 控制主内容区显示哪个页面
- `showLyricsView`: 独立状态，覆盖activeView显示歌词
- 歌词按钮切换 `showLyricsView = !showLyricsView`

## 响应式设计

### 窗口尺寸适配
- **最小宽度**: 800px (保证基本可用性)
- **窄窗口** (< 900px): 导航栏折叠为图标模式（仅显示图标）
- **紧凑模式**: 播放控制栏隐藏部分文本，显示图标为主

### 组件适应性
- 歌曲列表：列宽自适应，过窄时隐藏次要列
- 歌词视图：字体大小根据窗口宽度微调
- 播放控制栏：宽度不足时隐藏播放列表按钮

## Win11设计实现

### 视觉特性
1. **亚克力效果**: 导航栏、播放控制栏使用背景模糊
2. **圆角设计**: 所有按钮、卡片、窗口使用8px圆角
3. **流畅动画**:
   - 页面切换: 淡入淡出 + 横向滑动
   - 歌词滚动: 平滑滚动过渡
   - 按钮交互: 轻微缩放效果
4. **阴影层次**: 卡片悬浮阴影，导航栏深度阴影

### 深色/浅色主题
**v1.0**: 固定深色主题（Win11风格）
**v1.1**: 实现主题切换（深色/浅色/跟随系统）

## 交互流程示例

### 播放歌曲流程
1. 用户在歌曲列表双击歌曲
2. 应用播放该歌曲，底部控制栏更新信息
3. 点击歌词按钮 🎵 切换至歌词视图
4. 歌词随播放进度自动滚动
5. 点击返回按钮或再次点击歌词按钮返回列表

### 管理歌单流程
1. 导航到"歌单管理"
2. 点击"+ 新建歌单"，输入名称
3. 返回"全部歌曲"，选择多首歌曲
4. 右键菜单选择"添加到歌单"
5. 选择目标歌单完成添加
